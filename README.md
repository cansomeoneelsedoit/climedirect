# Clime Direct

Internal ASX investment platform — Clever Clime methodology.

## Features
- Intrinsic Value calculator (Equity Multiplier, Tiers 1–5)
- 268 ASX stocks with March 2026 valuation data
- Adjusted Valuation with live override inputs
- Portfolio Signals (Buy / Watch / Hold / Sell)
- Technical Analysis (RSI, MACD, Bollinger)
- AI Assistant (Claude, Clever Clime system prompt)
- FMP live data feed (optional API key)
- FactSet workflow guide

## Local
```
npm install
npm start
# Open http://localhost:3000
```

## Deploy
Hosted on Railway — auto-deploys on push to `main`.
